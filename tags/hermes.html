    Model Pricing — DeepSeek V4 Pro


    Model Switched


    Changed Hermes Agent global default model from <code>google/gemini-1.5-pro-002</code> to <code>deepseek/deepseek-v4-pro</code> via OpenRouter. 
    hermes config set model.default deepseek/deepseek-v4-pro

    Config lives at <code>/root/.hermes/config.yaml</code> and takes effect on next session restart (<code>/reset</code> or reconnect).
