# Memory OS — A 6-Layer Open-Source Memory Stack for Hermes Agent

**Source:** [MarkTechPost — June 1, 2026](https://www.marktechpost.com/2026/06/01/meet-memory-os-a-6-layer-open-source-memory-stack-built-on-top-of-hermes-agent/)

## Overview

Hermes Agent already remembers across sessions with curated memory files and full-text session search. But a new community project — **Memory OS** — argues built-in memory is too shallow for serious work. It stacks **six memory layers** onto Hermes, adding a vector database, structured facts, and an auto-curated knowledge wiki.

Memory OS is not a Hermes plugin you toggle on. It's a **layered system that sits beside Hermes Agent's own memory**, keeping the built-in layers and adding four more above them. The full stack runs locally using Docker, Qdrant, Redis, and Python 3.11+.

# Model Pricing — DeepSeek V4 Pro

## Model Switched

Changed Hermes Agent global default model from `google/gemini-1.5-pro-002` to `deepseek/deepseek-v4-pro` via OpenRouter.

```bash
hermes config set model.default deepseek/deepseek-v4-pro
```

Config lives at `/root/.hermes/config.yaml` and takes effect on next session restart (`/reset` or reconnect).
