Overview


Hermes Agent already remembers across sessions with curated memory files and full-text session search. But a new community project — <strong>Memory OS</strong> — argues built-in memory is too shallow for serious work. It stacks <strong>six memory layers</strong> onto Hermes, adding a vector database, structured facts, and an auto-curated knowledge wiki.
Memory OS is not a Hermes plugin you toggle on. It's a <strong>layered system that sits beside Hermes Agent's own memory</strong>, keeping the built-in layers and adding four more above them. The full stack runs locally using Docker, Qdrant, Redis, and Python 3.11+.
