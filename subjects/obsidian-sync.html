# Syncthing Troubleshooting Guide

## Setup
- **Server:** `100.114.52.106:22000`, device ID `T7HL47R-...`
- **Windows (mastermind):** `100.74.69.32`, device ID `AFRQORX-...`
- **Folder:** `obsidian-vault` → server: `/root/Documents/Obsidian Vault`, Windows: `C:\Obsidian Vault`

## Common Issues & Fixes

### 1. NordVPN Blocks Connection
**Symptom:** Tailscale shows "active" but Syncthing says "Disconnected"
**Fix:** Disable NordVPN, or set up split tunneling to exclude `syncthing.exe` and `tailscale.exe`

### 2. Port 22000 Blocked
**Symptom:** `lastSeen: never`, `connected=False`
**Fix:** Windows Defender Firewall → Inbound Rule → TCP 22000 → Private networks only

### 3. QUIC Listener Warning
**Symptom:** `quic://0.0.0.0:22000: bind: Only one usage...`
**Fix:** Settings → Connections → blank out QUIC Listen Addresses. Harmless otherwise.

### 4. Case Conflict (Projects vs projects)
**Symptom:** "remote uses different upper or lowercase characters"
**Fix:** Consolidate into one case (lowercase preferred). Merge files, delete duplicate directory.

### 5. GUI Not Accessible
**Fix:** Bind to `127.0.0.1:8384` for local-only access. Use `0.0.0.0:8384` for remote.
